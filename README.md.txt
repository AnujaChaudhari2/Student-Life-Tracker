# 🎓 Student Life Tracker

A Streamlit-based dashboard to track:

- 📚 Study Hours
- 💰 Expenses
- ⚡ Productivity
- 🎯 Monthly Goals
- 🔥 Study Streak
- 📅 Heatmap Calendar

## Features

- Weekly Analytics
- Monthly Analytics
- Goal Tracking
- Delete Functionality
- Beautiful Dark UI

## Tech Stack

- Python
- Streamlit
- Pandas
- Plotly

## Run Locally

```bash
streamlit run app.py
